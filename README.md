# Data-Cleaning-and-visualisation-in-Tableau

Aim:
Analysis on Sales data in Excel and Data visualization in Tableau .


Tools:
Excel and Tableau.


Project Description:
The Scope of the Project is data cleaning in Excel before uploading into Tableau.Data cleaning is a critical step in the data analysis process as inaccurate or inconsistent data can lead to incorrect conclusions and decisions. Some common tasks involved in data cleaning include removing duplicates, correcting spelling errors, fixing capitalization, removing leading or trailing spaces, replacing values, checking for missing data, fixing date and time formats, formatting data, and checking for errors.
